okinawarb/meetups
=======
Okinawa.rb Meetup!の案内と、活動記録を掲載したウェブページを管理しているリポジトリです。

内容は[Okinawa.rb Meetups](http://ruby.okinawa/meetups/)から確認できます。

以下のような感じで、ローカルから確認できます。

```
$ bundle install
$ bundle exec jekyll serve -w
$ open http://localhost:4000/meetups/
```
